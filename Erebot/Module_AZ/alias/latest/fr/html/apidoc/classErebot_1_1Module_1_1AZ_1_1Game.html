Erebot\Module\AZ\Game::isValidWord (   $word)
protected

Checks whether some text is a valid word for the current game.

This method checks whether the given text contains something that is recognized as a word in the current range, and part of the currently active wordlists.

Parameters
string$wordSome "word" to test.
Return values
mixednull is returned when the given $word does not look like a word (eg. "#$!@") or is outside the game's current range. false is returned when this word is not part of a currently active wordlist. Otherwise, the word is returned the way it is written in the list (which may include case or accentuation variations).

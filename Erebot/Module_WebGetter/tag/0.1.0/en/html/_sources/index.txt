Welcome to the documentation for Erebot_Module_WebGetter!
=========================================================

Erebot_Module_WebGetter is a module for `Erebot`_ that fetches resources
from Internet using an HTTP GET or POST method.
The content is then parsed to extract some pieces of information using
`XPath expressions`_ [#]_.

Welcome to the documentation for Erebot_Module_WebGetter!¶

Erebot_Module_WebGetter is a module for Erebot that fetches resources from Internet using an HTTP GET or POST method. The content is then parsed to extract some pieces of information using XPath expressions [1].

Contents:

Current status on http://travis-ci.org/:

[1]XPath is a W3C recommendation designed as a language for addressing parts of an XML document.

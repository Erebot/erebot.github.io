Compatible modules¶

This page lists several modules which are known to be compatible with Erebot, divided into two categories:

  1. Supported modules
  2. Third-party modules

Supported modules¶

This section contains a list of supported modules:

  • AZ module
  • Admin module
  • AutoConnect module
  • AutoIdent module
  • AutoJoin module
  • Countdown module
  • CtcpResponder module
  • GoF module
  • Helper module
  • IrcConnector module
  • IrcTracker module
  • LagChecker module
  • Math module
  • MiniSed module
  • PhpFilter module
  • PingReply module
  • RateLimiter module
  • Roulette module
  • ServerCapabilities module
  • TV module
  • TriggerRegistry module
  • Uno module
  • WatchList module
  • WebGetter module
  • Wordlists module

The source code for supported modules can be found at https://github.com/Erebot/ (look for repositories whose name starts with Module_). See each module’s individual documentation for additional information.

Third-party modules¶

This sections contains a list of third-party modules:

No third-party module has been submitted yet.

Contact us at erebot@erebot.net if you created a module and would like to see it listed here.

static Erebot\Module\Wordlists\Base::isWord (   $word)
static

Filters non-words out.

Only texts that passed this filtering step are considered as propositions for the game.

Parameters
string$wordA possible "word" to test.
Return values
boolTRUE if the given $word really is a word, FALSE otherwise.
Note
This method uses a rather broad definition of what is a word. In particular, it accepts sequences of (alphanumeric and other) characters separated by a single space (eg. "Fo'o. B4-r_").

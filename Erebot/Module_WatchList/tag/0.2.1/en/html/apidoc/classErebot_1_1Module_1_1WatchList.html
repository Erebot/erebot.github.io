Detailed Description

A module that can be used to manage the bot's IRC WATCH list.

This module adds support for the WATCH/ISON commands to Erebot. It can be used to monitor (dis)connections of some IRC users.

Attention
The WATCH command is an extension to the official IRC protocol and is supported only by some IRC networks. Most IRC servers that implement it have strict restrictions on the number of users someone may track using this command. The ISON command however is safe (and is part of the official IRC protocol). If you plan to track a lot of users, don't use this module.
Note
In the future, this module may use the most appropriate command to monitor (dis)connections, depending on the number of nicks it has to track.
